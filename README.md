## Mise en ligne GitHub Pages
1. Crée un nouveau dépôt GitHub.
2. Envoie `index.html` et le dossier `images`.
3. Va dans `Settings` > `Pages`.
4. Source : `Deploy from a branch`.
5. Branch : `main` / folder : `/root`.
